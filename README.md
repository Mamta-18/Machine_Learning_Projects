# 🤖 Machine Learning Projects

This repository contains a collection of end-to-end **Machine Learning projects** implemented using Python.  
The projects are organized into two main categories: **Classification** and **Regression**, showcasing both predictive modeling techniques.

---



## 🔍 Project Categories


### 📈 Regression Projects
Regression models are used to predict **continuous numerical values**.

| Project | Description |
|--------|-------------|
| Big Mart Sales Prediction | Forecasts sales of retail items across different outlets |
| Gold Price Prediction | Predicts future gold prices using historical market data |

---
### 🧪 Classification Projects
Classification models are used to predict **discrete categories / labels**.

| Project | Description |
|--------|-------------|
| Heart Disease Prediction | Predicts the presence of heart disease using clinical data |

---

## 🛠️ Tech Stack Used
- Python
- NumPy & Pandas
- Matplotlib & Seaborn
- Scikit-Learn
- Jupyter Notebook

✔ Performance Metrics & Reporting
